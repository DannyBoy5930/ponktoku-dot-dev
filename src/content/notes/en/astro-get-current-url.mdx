[Astro.url](https://docs.astro.build/en/reference/api-reference/#astrourl") returns the current page URL from the Request object.
The return value is a [URL](https://developer.mozilla.org/en-US/docs/Web/API/URL) object which contains properties like pathname and origin.

```js
const currentPath = Astro.url.pathname;
```

Useful when you need to highlight navigation links based on current page:

```jsx
<a href="/me" class={currentPath === "/me" ? "active" : ""}>
  About Me
</a>
```

## Splitting nested URL paths

Use [String.split()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split)
to split an input string into an array of substrings. It's split based on a specified operator such as a slash (`/`).

```js
const currentPath = Astro.url.pathname; // "/foo/bar"
const firstPath = currentPath.split("/")[1];
console.log(firstPath); // returns 'foo'

currentPath.split("/")[0]; // returns '/'
currentPath.split("/")[2]; // returns 'bar'
```

Access a specific element of the resulting array by using the index number either at the end like in `split()[2]`.
